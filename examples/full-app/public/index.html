Full App

End-to-end MiiaJS example. Drizzle + PostgreSQL, JWT auth with roles, Swagger UI, CORS, request logging - all wired together in ~400 lines of application code.

What's inside

  • Drizzle ORM with postgres driver and onReady() admin seed
  • JWT auth (@miiajs/auth/jwt) with Local and JWT strategies
  • Role-based access via RolesGuard + @Roles('admin')
  • Full OpenAPI 3.1 spec generated from decorators
  • CORS, request-id, response-time, request logger middleware
  • Zod validation on body / query / params
