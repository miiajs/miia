# @miiajs/rate-limit

Fixed-window rate limiting for MiiaJS - `RateLimitGuard` with `@RateLimit`/`@SkipRateLimit` decorators, `rateLimit` perimeter middleware, pluggable stores, draft-6 `RateLimit-*` headers.

## Installation

```bash
npm install @miiajs/rate-limit
```
