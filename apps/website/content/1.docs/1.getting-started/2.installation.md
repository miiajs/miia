## Scaffold with CLI (recommended)

The fastest way to start a new project is with the [`@miiajs/cli`](/docs/packages/cli):

::code-group

```bash [bun]
bunx @miiajs/cli new my-app
```

```bash [npm]
npx @miiajs/cli new my-app
```

```bash [pnpm]
pnpm dlx @miiajs/cli new my-app
```

```bash [yarn]
yarn dlx @miiajs/cli new my-app
```

::

```bash
cd my-app
```

The CLI will prompt you to choose a runtime (Bun, Deno, or Node.js), generate the project structure, install dependencies, and configure TypeScript - all in one step. Then run:

::code-group

```bash [bun]
bun run dev
```

```bash [npm]
npm run dev
```

```bash [pnpm]
pnpm dev
```

```bash [yarn]
yarn dev
```

::

If you prefer to set things up manually, follow the steps below.

## Manual setup

### Core package

::code-group

```bash [bun]
bun add @miiajs/core
```

```bash [npm]
npm install @miiajs/core
```

```bash [pnpm]
pnpm add @miiajs/core
```

```bash [yarn]
yarn add @miiajs/core
```

::

## Server adapters

MiiaJS exposes a standard `fetch` handler. Choose a server adapter for your runtime:

### Bun & Deno

No additional package needed. `app.listen()` auto-detects the runtime.

### Node.js

::code-group

```bash [bun]
bun add @miiajs/node-server
```

```bash [npm]
npm install @miiajs/node-server
```

```bash [pnpm]
pnpm add @miiajs/node-server
```

```bash [yarn]
yarn add @miiajs/node-server
```

::

### uWebSockets.js

::code-group

```bash [bun]
bun add @miiajs/uws-server uWebSockets.js@uNetworking/uWebSockets.js#v20.68.0
```

```bash [npm]
npm install @miiajs/uws-server uNetworking/uWebSockets.js#v20.68.0
```

```bash [pnpm]
pnpm add @miiajs/uws-server uNetworking/uWebSockets.js#v20.68.0
```

```bash [yarn]
yarn add @miiajs/uws-server uNetworking/uWebSockets.js#v20.68.0
```

::

## TypeScript configuration

MiiaJS requires ES2025 for TC39 decorator support:

```json
{
  "compilerOptions": {
    "target": "ES2025",
    "module": "nodenext",
    "verbatimModuleSyntax": true
  }
}
```

For the full list of packages, see [Packages](/docs/packages).
