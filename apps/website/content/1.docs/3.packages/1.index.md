MiiaJS is modular - install only what you need.

## Server adapters

| Package | Description |
| --- | --- |
| [`@miiajs/node-server`](/docs/packages/node-server) | Node.js HTTP server adapter |
| [`@miiajs/uws-server`](/docs/packages/uws-server) | uWebSockets.js HTTP server adapter |

Bun and Deno are supported out of the box - no adapter needed.

## Static files

| Package | Description |
| --- | --- |
| [`@miiajs/serve-static`](/docs/packages/serve-static) | Static file serving with Range, ETag, charset, SPA fallback |

## Auth

| Package | Description |
| --- | --- |
| [`@miiajs/auth`](/docs/packages/auth) | Auth primitives - `AuthProvider`, `AuthGuard`, token extractors |
| [`@miiajs/jwt`](/docs/packages/jwt) | Injectable JWT sign/verify service (jose wrapper) |

## Database

| Package | Description |
| --- | --- |
| [`@miiajs/drizzle`](/docs/packages/drizzle) | Drizzle ORM integration (PostgreSQL, MySQL, SQLite) |
| [`@miiajs/mongoose`](/docs/packages/mongoose) | MongoDB integration via Mongoose with injectable models |
| [`@miiajs/papr`](/docs/packages/papr) | MongoDB integration via Papr |

## Messaging

| Package | Description |
| --- | --- |
| [`@miiajs/messaging`](/docs/packages/messaging) | Decorator-driven event bus with retry, DLQ, pluggable transports |
| [`@miiajs/messaging-redis`](/docs/packages/messaging/redis) | Redis Streams transport for `@miiajs/messaging` |
